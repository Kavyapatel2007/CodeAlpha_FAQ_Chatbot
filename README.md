# AI FAQ Chatbot

## Description
This project is an AI-powered FAQ chatbot developed using Python, Flask, Scikit-learn, and NLP.

## Features
- Answer FAQs
- Cosine Similarity Matching
- NLP-based text processing
- Responsive chat interface

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- HTML
- CSS
- JavaScript

## How to Run
1. Install requirements:
pip install -r requirements.txt

2. Run:
python app.py

3. Open:
http://127.0.0.1:5000
